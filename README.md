# 👩‍💻 Sonia Nazeer - Developer Portfolio

This is a fully responsive personal portfolio website built using **React**, **Bootstrap**, and **CSS**. It features clean design, smooth responsiveness, and a layout designed to highlight your profile, technical skills, professional experience, and portfolio projects.

---

## 🚀 Live Preview

https://personal-portfolio-react-theta-jet.vercel.app/

---

## 📌 Features

- ⚛️ Built with React (Create React App)
- 💠 Fully responsive layout using Bootstrap and custom CSS
- 👩‍💻 Sections: Profile, About, Skills, Experience, Projects, and Contact
- 🎬 Video preview for portfolio projects
- 📥 Downloadable CV
- 📧 Contact form (currently static)
- 🌐 Social media integration (GitHub, LinkedIn, Instagram)

---

## 🛠️ Technologies Used

- React
- Bootstrap 5
- CSS3
- HTML5
- Font Awesome Icons
- Google Fonts
